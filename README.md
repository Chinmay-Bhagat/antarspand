# AntarSpand

Personal blog on wearable health tech, biosignals, and biomedical engineering.

Live at [antarspand.com](https://antarspand.com)

## Running locally

```sh
npm install
npx astro dev --port 4321
```

## Building

```sh
npx astro build
```

## Stack

- [Astro 6](https://astro.build) — static site generator
- Markdown content via Astro Content Collections
- Deployed on GitHub Pages with a custom domain
